# Credit Tracker

Give consumers a 1-on-1 mirror of their credit card's real cost — daily
interest accrual, per-purchase installment breakdowns, and the real cost of
credit that banks don't surface. See `docs/prd.md` and `docs/user-journeys.md`
for full product context.

**Status**: walking skeleton — Colombia market only, calculation engine and
one end-to-end API flow proven. US market, frontend, and deployment pipeline
are next.

## Quick start (local)

```bash
cd backend
pip install -e ".[dev]" --break-system-packages   # or use a venv instead
pytest tests/ -v
```

## Run with Docker (backend + Postgres)

```bash
docker compose up --build
# then, once Postgres is healthy:
docker compose exec backend alembic upgrade head
```

API docs (auto-generated by FastAPI) will be at `http://localhost:8000/docs`.

## Try it

```bash
# Create a Colombian credit card
curl -X POST http://localhost:8000/api/v1/internal/products \
  -H "Content-Type: application/json" \
  -d '{"market": "CO", "institution_name": "Banco de Prueba", "credit_limit": 5000000, "ea_rate": 0.36, "day_count_basis": 365}'

# Add a purchase deferred into 6 installments (use the product id returned above)
curl -X POST http://localhost:8000/api/v1/internal/products/{product_id}/purchases \
  -H "Content-Type: application/json" \
  -d '{"amount": 600000, "currency": "COP", "purchase_date": "2026-07-01", "n_installments": 6}'

# Get the real principal/interest breakdown (use the purchase id returned above)
curl http://localhost:8000/api/v1/internal/products/{product_id}/purchases/{purchase_id}/schedule
```

## Project structure

```
├── app/
│   ├── main.py
│   ├── core/            # config, DB session
│   ├── products/        # models + schemas
│   ├── calculations/     # the core engine — rates.py, amortization.py
│   ├── accounts/         # reserved (auth, not built yet)
│   ├── integrations/     # reserved (third-party API, not built yet)
│   └── api/v1/{internal,public}/
├── migrations/           # Alembic
├── tests/{unit,integration}/
├── Dockerfile
└── pyproject.toml
docker-compose.yml
docs/
├── prd.md
└── user-journeys.md
```
